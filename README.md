완전 초보자를 위한 한국어 백과사전급 불렛저널(Bullet Journal) 매뉴얼입니다.

## 특징

- **14장 + 부록 3종** 구성의 단일 마크다운 파일
- 완전 초보자 기준 — 모든 용어를 처음 등장 시 정의
- 한국 맥락 (원화, 한국 제품, 한국 공휴일, 한식 등) 기준 예시
- 25개 출처 도메인에서 100개 이상의 참고 자료 링크 제공
- 12장(첫 30일 가이드)만 읽어도 바로 시작 가능
- 목차 전 장에 하위 섹션 직접 링크 제공 (118개 앵커 검증 완료)
- 라이더 캐롤의 **의도(Intention)** 철학을 전용 섹션(6.1)으로 심화 수록

## 목차 요약

| 장 | 제목 | 내용 |
|----|------|------|
| 1장 | 불렛저널이란 무엇인가? | 탄생 배경, 핵심 철학(의도·마음챙김·생산성), 자가진단 |
| 2장 | 시작하기 전에: 준비물 | 노트+펜 추천, 한국 구매 가이드 |
| 3장 | 래피드 로깅 | 기호 체계, 무드 불릿(=), 실전 예시, 잘못된 예시 |
| 4장 | 기본 컬렉션 4가지 | 인덱스, 퓨처/먼슬리/데일리 로그 |
| 5장 | 커스텀 컬렉션 8종 | 습관 트래커, 가계부, 독서, 무드 등 |
| 6장 | 자기계발과 감정 관리 | **의도(Intention) 심화**, 회고, 감정 기록, 목표 분해, 마인드 덤프 |
| 7장 | 마이그레이션 | 일일/월간/연간 마이그레이션, 플로우차트 |
| 8장 | 나에게 맞는 스타일 | 미니멀/데코/하이브리드, 스타일 진단 |
| 9장 | 디지털 도구와 함께 쓰기 | 아날로그+디지털 조합, 추천 앱 |
| 10장 | 흔한 실수와 해결법 | 시작/유지 단계 실수, 포기 대처법 |
| 11장 | 장기 지속 전략 | 번아웃 방지, 리뷰 체크리스트, 권태기 극복 |
| 12장 | 첫 30일 실전 가이드 | 1일차~30일차 단계별 온보딩 |
| 13장 | 치트시트 | 기호, 구조, 마이그레이션 한 페이지 요약 |
| 14장 | FAQ | 자주 묻는 질문 17개 |
| 부록 | 추천 도서 / 커뮤니티 / 전체 출처 | 도서 7권, 커뮤니티 목록, 출처 26개 |
